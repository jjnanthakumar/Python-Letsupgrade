### **Q**: _check the "is" operator on the  float and string write you findings?_

**A**:

      For strings :

       >>> m = 'abc'
       >>> n = 'abc'
       >>> m is n
       True
       

       For float : 
       
       >>> a,b = 10.12, 10.12
       >>> a is b
       True
